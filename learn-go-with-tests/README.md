# Learn Go with Tests

[learn go with tests](https://quii.gitbook.io/learn-go-with-tests)

When starting a go new project, initialize by:

```shell
go mod init example.com/modulename
```

To build:

```shell
go build .
```

To build and run:

```shell
go run .
```

To clean build files:

```shell
go clean
```

To run tests:

```shell
go test
```

To run benchmarks:

```shell
go test -bench=.
```

To check test coverage:

```shell
go test -cover
```
